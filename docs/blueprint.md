# **App Name**: LeetMastery

## Core Features:

- Firebase Authentication: Secure user authentication using Firebase Auth.
- Home Dashboard - Daily Questions: Display 4-5 LeetCode questions with links, topic references, and personal notes for quick access.
- Topics Management: Create, view, edit, and delete topics with descriptions, notes (text, pseudocode, snippets), and associated questions.
- Topic Questions: Associate LeetCode questions to topics, including links, titles, difficulty, status buckets (To-Do, Solved, Repeat, Important, Tricky), personal notes, and last solved date.  Enable filtering by bucket.
- AI Problem Selection Tool: Suggest LeetCode problems to the user based on performance by using the bucket history to identify weaker areas to target.
- Firestore Integration: Use Firestore to store user data, including daily questions, topics, questions, notes and configurations.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to promote focus and tranquility.
- Background color: Light gray (#F5F5F5) to minimize distraction.
- Accent color: Soft green (#A5D6A7) for positive reinforcement when marking a question as solved.
- Body font: 'Inter' sans-serif, for a clean, modern, objective reading experience.
- Headline font: 'PT Sans' sans-serif, providing warmth and personality while maintaining a modern look.
- Use minimalist icons for topics and question statuses.
- Clean and spacious layout optimized for readability and focus.
- Subtle animations when transitioning between sections and updating question statuses.
